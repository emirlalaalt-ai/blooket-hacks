# Blooket Hacks

> Use at your own risk. This may violate Blooket's Terms of Service.

## How to Use

1. Go to [blooket.com](https://www.blooket.com) and join a game.
2. Open Developer Tools — press `F12` (or `Cmd+Option+I` on Mac).
3. Click the **Console** tab.
4. Paste whichever script you want and hit **Enter**.

That's it.
